# 🧣 《[羊毛袜里的暖意](https://mrdsonle.github.io/)》

北方的冬天总是来得猝不及防，第一场雪落下时，巷口的老槐树还没来得及落尽叶子。李婶的针织摊支在单元楼门口，竹篮里堆着五颜六色的毛线团，织好的围巾和手套挂在树枝上，像一串串彩色的风铃。

老周每天下班都会绕过来，要么选一团藏蓝色的毛线，要么带走一双刚织好的羊毛袜。他总说“给老伴儿带的”，李婶就笑着调侃“又哄老婆子开心啊”，手里的织针却不停，织针碰撞发出的嗒嗒声，像冬日里的小鼓点。

没人知道他们是一对，只记得去年冬天老周摔了一跤，腿骨骨折，李婶的针织摊就挪到了他家楼下。每天早上李婶都会端着一碗热粥上去，晚上再把织好的袜子放在他家门口的台阶上。老周康复后，每天都会帮李婶把针织摊搬回家，竹篮的把手被他磨得发亮。

上周降温，李婶的手冻得裂了好几道口子，老周就把自己的羊毛手套给她戴上，说“我糙皮厚肉的，没事”。李婶红着眼眶把织了一半的围巾绕在他脖子上，说“等织好了，就不用冻着了”。围巾上的花纹是老周最喜欢的棋盘格，针脚歪歪扭扭，却比任何名牌围巾都温暖。

昨天我路过单元楼门口，看见老周坐在小马扎上，帮李婶绕毛线，阳光透过树枝的缝隙洒下来，落在他们身上，像一层金色的纱。李婶的织针在毛线间穿梭，偶尔会停下来，把暖手宝塞进老周的口袋里，老周就笑着把一颗剥好的糖放进她嘴里。

现在李婶的针织摊上，多了一个用毛线织成的小坐垫，上面绣着两个歪歪扭扭的字：“周”“李”。风一吹，坐垫轻轻晃动，像在诉说着他们的故事。

金句： 最动人的爱情，是在寒冷的日子里，把彼此的暖意，织进一针一线里。
